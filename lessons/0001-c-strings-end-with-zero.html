Find the end of a C string

Goal: implement your own ft_strlen.

What you need

  • A C string is a sequence of characters followed by '\0'.
  • s[0] is its first character, s[1] is its second, and so on.
  • ft_strlen counts the characters before '\0'. It does not count '\0'.
index:       0     1     2     3
character:  'c'   'a'   't'   '\0'

The string length is 3.
