Test for a letter

Goal: implement ft_isalpha.

What you need

  • In this project, a letter is one of A–Z or a–z.
  • The function receives an int and returns 1 for a letter or 0 otherwise.
  • Use comparisons: c >= 'A' && c <= 'Z' describes the uppercase range.

Exercise

Complete ft_isalpha. Do not use the library function isalpha.
